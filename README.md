# 簡易電子琴 Beep Player

## 專案簡介

這是一個使用 C# Windows Forms 製作的簡易電子琴程式。本專案透過 Windows 系統提供的 `Beep` 音效功能，讓使用者可以點擊按鈕播放不同音高的音符，並可從下拉式選單選擇內建旋律進行播放。

本程式模擬簡易電子琴介面，畫面中提供多個音符按鈕，例如 Do、Re、Mi、Fa、Sol、La、Si、高音 Do。使用者點擊不同按鈕時，程式會播放對應頻率的聲音，達到基本電子琴的效果。

除了單音播放功能外，程式也新增了「播放簡單旋律」功能。使用者可以透過下拉式選單選擇內建歌曲，例如「生日快樂歌」或「超級瑪利歐」，再按下播放按鈕，即可自動播放對應旋律。

為了避免播放旋律時使用者同時按下單音按鈕，造成聲音重疊或播放混亂，程式在播放旋律期間會暫時停用 Do、Re、Mi、Fa、Sol、La、Si、高音 Do 等音符按鈕，待旋律播放完成後再自動恢復按鈕功能。

---

## 執行說明與主要功能

### 1. 單音播放

使用者可以點擊畫面上的音符按鈕播放對應音高。

目前提供的音符包含：

- Do
- Re
- Mi
- Fa
- Sol
- La
- Si
- 高音 Do

每個音符按鈕都會對應一個固定頻率。按下按鈕後，程式會透過 `Beep` 函式播放短音，讓使用者可以像操作簡易電子琴一樣播放單一音符。

---

### 2. 播放簡單旋律

程式內建數首簡單旋律，使用者可以從下拉式選單中選擇想播放的歌曲。

目前支援的旋律包含：

- 生日快樂歌
- 超級瑪利歐

選擇歌曲後，按下「播放旋律」按鈕，程式會依照預設的音符頻率與播放時間，自動播放完整旋律。

---

### 3. 播放旋律時停用單音按鈕

當程式正在播放內建旋律時，畫面上的單音按鈕會暫時停用，使用者無法同時按下 Do、Re、Mi 等音符按鈕。

此設計可以避免旋律播放過程中又觸發其他單音，導致聲音互相重疊或播放效果混亂。

旋律播放完成後，程式會自動恢復所有單音按鈕，使用者可以繼續進行單音播放。

---

### 4. 視窗縮放控制項調整

程式會記錄視窗初始大小與控制項位置。當使用者調整視窗大小時，介面中的按鈕與控制項會根據視窗比例進行縮放與移動，使畫面保持較好的配置。

此功能讓程式在不同視窗大小下仍能維持較整齊的版面。

---

### 5. 關閉確認提示

當使用者關閉程式時，系統會跳出確認視窗詢問是否確定關閉，避免使用者誤按關閉按鈕。

如果使用者選擇取消，程式會停止關閉動作並繼續執行；如果使用者選擇確定，程式才會結束。
